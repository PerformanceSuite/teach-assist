# Quick Start Guide

Get productive with TeachAssist in under 5 minutes.

## Step 1: Upload Your First Source (1 minute)

1. Click **Upload Sources** on the home page
2. Select a PDF or Word document (curriculum standards work great)
3. Add a descriptive title like "8th Grade Science Standards"
4. Wait for processing (usually under 30 seconds)

**Pro tip:** Start with your core curriculum standards - they're the foundation for everything else.

## Step 2: Ask a Question (1 minute)

1. Navigate to **Chat** or press `Cmd+J`
2. Type a question about your uploaded content
3. Press Enter

**Try this:** "What are the key learning objectives in this document?"

You'll get an answer that cites specific sections from your source.

## Step 3: Consult Your Council (2 minutes)

1. Navigate to **Inner Council**
2. Choose an advisor:
   - **Standards Guardian** for alignment checks
   - **Differentiation Architect** for accommodation ideas
   - **Assessment Designer** for rubric help
   - **Curriculum Strategist** for big-picture planning
3. Provide context (what you're working on)
4. Ask your question

**Try this:** Ask the Standards Guardian "How does this lesson align with the uploaded standards?"

## Step 4: Explore Keyboard Shortcuts (1 minute)

| Shortcut | Action |
|----------|--------|
| `Cmd+K` | Command Palette |
| `Cmd+J` | Toggle Chat |
| `Cmd+/` | Help Center |
| `Cmd+U` | Upload Document |

Press `Cmd+K` to see all available commands.

## What's Next?

- Upload more sources (lesson plans, IEPs, teaching frameworks)
- Try different advisors for various perspectives
- Read about [privacy protections](/help/privacy-first) to understand data handling
- Learn about [narrative synthesis](/help/narrative-overview) for report card comments

## Need Help?

- Press `Cmd+/` anytime to open Help Center
- Search for specific topics
- Check contextual tips that appear as you navigate

You're ready to go. Start by uploading something you're working on right now.
